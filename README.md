# Time Dial

A simple & small jQuery plugin for creating an animated time dial.
[See it in action](http://foo.bar)

## Usage

```
<div id="my-dial"></div>
```
```
<script src="jquery.min.js"></script>
<script src="timedial.min.js"></script>
<script>
  $('div#my-dial').timedial();
</script>
```
